# 资料库看板

---

## 1. 全库文档总览

```dataview
TABLE 文档类型 AS "类型", 状态 AS "状态", 编制人 AS "编制人", 编制日期 AS "日期"
FROM ""
WHERE 文档类型 != null
SORT 编制日期 DESC
LIMIT 20
```

---

## 2. 按文档类型分布

```dataview
TABLE length(rows) AS "数量"
FROM ""
WHERE 文档类型 != null
GROUP BY 文档类型
SORT length(rows) DESC
```

## 5. 按编制人统计

```dataview

FROM ""
WHERE 编制人 != null
GROUP BY 编制人
SORT length(rows) DESC
```
