# 反洗钱 (AML) 规则与流程

| 项目 | 内容 |
|------|------|
| 文档编号 | COMP-AML-2026-001 |
| 版本 | V2.3 |
| 编制日期 | 2026-04-01 |
| 编制人 | 刘建国（合规总监） |
| 法规依据 | 《反洗钱法》《金融机构大额交易和可疑交易报告管理办法》 |

---

## 1. 总则

### 1.1 适用范围

公司所有保险业务，包括但不限于：
- 人身保险（寿险、健康险、意外险）
- 财产保险（车险、企财险、责任险）
- 再保险业务

### 1.2 基本原则

| 原则 | 说明 |
|------|------|
| 了解你的客户 (KYC) | 核实客户身份，了解交易目的 |
| 大额报告 | 单笔或当日累计超过规定金额须报告 |
| 可疑报告 | 交易异常或与客户身份不符须报告 |
| 持续监控 | 对存量客户持续进行风险监控 |
| 记录保存 | 交易记录保存不少于5年 |

## 2. 客户身份识别

### 2.1 识别场景

| 场景 | 识别要求 | 系统措施 |
|------|----------|----------|
| 新客户投保 | 必须核实身份 | 身份证OCR+联网核查 |
| 大额投保 | 加强身份识别 | 人脸识别+电话回访 |
| 受益人识别 | 识别最终受益人 | 受益人信息采集 |
| 代理人识别 | 核实代理人资质 | 代理人资格证验证 |

### 2.2 客户风险等级

| 等级 | 标准 | 监控频率 |
|------|------|----------|
| 低风险 | 普通个人客户，年保费 ≤ 5万 | 年度复审 |
| 中风险 | 年保费 5-50万，或高风险职业 | 半年复审 |
| 高风险 | 年保费 > 50万，或政治敏感人物(PEP) | 季度复审 |
| 禁止类 | 涉恐名单、制裁名单 | 禁止交易 |

### 2.3 政治敏感人物 (PEP) 识别

| 类型 | 识别方式 | 措施 |
|------|----------|------|
| 国内PEP | 自有名单库匹配 | 高级管理层审批 |
| 国际PEP | 第三方名单库(Watchlist) | 反洗钱岗审批 |
| PEP关联人 | 知识图谱分析 | 同PEP处理 |

## 3. 大额交易报告

### 3.1 报告标准

| 交易类型 | 报告阈值 | 报告时限 |
|----------|----------|----------|
| 现金缴费 | 单笔 ≥ 5万元人民币 | 交易发生后5个工作日内 |
| 转账缴费 | 单笔 ≥ 20万元人民币 | 交易发生后5个工作日内 |
| 外币缴费 | 单笔 ≥ 1万美元等值 | 交易发生后5个工作日内 |
| 退保/给付 | 单笔 ≥ 5万元人民币 | 交易发生后5个工作日内 |

### 3.2 系统实现

```
交易发生
    │
    ▼
实时金额检测
    │
    ├── 低于阈值 ──▶ 正常处理
    │
    └── 达到/超过阈值 ──▶ 自动生成大额交易报告
                              │
                              ▼
                         反洗钱岗审核
                              │
                              ├── 确认无误 ──▶ 上报人民银行
                              │
                              └── 需补充信息 ──▶ 联系业务部门补充
```

## 4. 可疑交易报告

### 4.1 可疑交易识别规则

| 规则编号 | 规则描述 | 触发条件 |
|----------|----------|----------|
| SAR-001 | 短期频繁投保退保 | 6个月内退保 ≥ 3次 |
| SAR-002 | 大额趸交后立即退保 | 趸交保费 ≥ 50万且30日内退保 |
| SAR-003 | 多账户集中缴费 | ≥ 3个不同账户为同一保单缴费 |
| SAR-004 | 投保金额与收入不符 | 年保费 > 年收入的50% |
| SAR-005 | 频繁变更受益人 | 1年内变更受益人 ≥ 3次 |
| SAR-006 | 使用他人账户缴费 | 缴费账户非投保人/被保险人 |
| SAR-007 | 异常地域交易 | 投保地与户籍地/工作地均不符 |
| SAR-008 | 匿名化尝试 | 使用现金且拒绝提供身份信息 |
| SAR-009 | 关联可疑交易 | 关联客户已被报告可疑交易 |
| SAR-010 | 整数金额交易 | 连续多笔整数金额（如10万、20万） |

### 4.2 可疑交易处理流程

```
规则触发
    │
    ▼
系统自动生成可疑交易预警
    │
    ▼
反洗钱岗初审（2个工作日内）
    │
    ├── 排除可疑 ──▶ 记录分析结论，关闭预警
    │
    └── 确认可疑 ──▶ 填写可疑交易报告
                         │
                         ▼
                    合规负责人复审
                         │
                         ▼
                    上报人民银行（发现之日起10个工作日内）
```

## 5. 名单监控

### 5.1 名单类型

| 名单类型  | 来源          | 更新频率 |
| ----- | ----------- | ---- |
| 制裁名单  | 联合国、OFAC、EU | 每日更新 |
| 涉恐名单  | 公安部         | 实时更新 |
| PEP名单 | 第三方数据商      | 每月更新 |
| 负面新闻  | 舆情监控        | 实时   |
| 内部黑名单 | 自有案件积累      | 实时   |

### 5.2 命中处理

| 命中类型 | 处理方式 |
|----------|----------|
| 制裁名单 | 立即冻结交易，上报合规官 |
| 涉恐名单 | 立即冻结交易，上报公安部门 |
| PEP名单 | 加强尽调，高级管理层审批 |
| 负面新闻 | 人工复核，判断关联性 |

## 6. 系统功能

### 6.1 交易监控

| 功能 | 说明 |
|------|------|
| 实时监控 | 交易发生时实时规则检测 |
| 批量扫描 | 每日批量扫描存量数据 |
| 链路分析 | 知识图谱关联分析 |
| 预警管理 | 预警生成、分配、处理、关闭全流程 |

### 6.2 报告生成

| 功能 | 说明 |
|------|------|
| 大额报告 | 自动生成，人工确认后上报 |
| 可疑报告 | 半自动填写，人工审核后上报 |
| 报告查询 | 按时间/类型/状态查询历史报告 |
| 报告统计 | 月度/季度/年度统计分析 |
