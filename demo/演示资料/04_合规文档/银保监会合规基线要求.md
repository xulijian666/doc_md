# 银保监会合规基线要求

## 1. 适用范围

本文档适用于公司所有保险业务系统，涵盖产品设计、销售、承保、理赔、保全全流程。

## 2. 产品合规要求

### 2.1 产品备案

| 要求 | 具体内容 | 系统落地措施 |
|------|----------|-------------|
| 产品审批 | 所有产品须经总精算师签字、合规部审核后报备 | 产品上架前增加合规审核工作流 |
| 条款审核 | 保险条款须符合《保险法》及监管规定 | 条款模板管理+法务审核节点 |
| 费率合理性 | 费率不得存在不正当竞争 | 费率审批流程+上下限管控 |
| 产品停售 | 停售须提前30日报备 | 停售申请+自动倒计时+客户通知 |

### 2.2 销售合规

| 要求 | 具体内容 | 系统落地措施 |
|------|----------|-------------|
| 如实告知 | 投保人须如实告知健康状况等 | 健康告知问卷+签字确认 |
| 销售误导禁止 | 不得夸大收益、隐瞒风险 | 话术库审核+AI实时检测 |
| 双录要求 | 一年期以上人身险须录音录像 | 双录系统对接+质检 |
| 犹豫期 | 不少于15天犹豫期 | 犹豫期自动计算+退保提示 |

## 3. 承保合规要求

### 3.1 反欺诈

| 要求 | 系统措施 |
|------|----------|
| 投保人身份核验 | 身份证OCR+联网核查+人脸识别 |
| 重复投保检测 | 同一被保险人多保单预警 |
| 高风险客户识别 | 风控评分模型+黑名单机制 |
| 可疑交易报告 | 大额投保自动上报 |

### 3.2 核保合规

| 要求 | 系统措施 |
|------|----------|
| 核保留痕 | 核保决策全记录，保留≥10年 |
| 核保时效 | 自动核保≤1分钟，人工核保≤24小时 |
| 拒保理由 | 拒保须给出明确理由 |

## 4. 理赔合规要求

### 4.1 理赔时效

| 环节 | 时限要求 | 系统管控 |
|------|----------|----------|
| 报案受理 | 收到报案后即时受理 | 系统自动受理 |
| 材料审核 | 材料齐全后5个工作日内作出核定 | 超时自动升级 |
| 赔款支付 | 达成赔偿协议后10日内支付 | 支付时效监控 |
| 拒赔通知 | 3日内发出拒赔通知书 | 自动通知模板 |

### 4.2 理赔材料

| 要求 | 系统措施 |
|------|----------|
| 材料清单 | 明确告知所需材料清单 |
| 材料一次性告知 | 补充材料须一次性告知 |
| 材料保存 | 原始材料扫描归档，保留≥5年 |

## 5. 数据合规要求

### 5.1 个人信息保护

| 要求 | 法规依据 | 系统措施 |
|------|----------|----------|
| 最小必要原则 | 《个人信息保护法》第6条 | 仅采集业务必需字段 |
| 知情同意 | 《个人信息保护法》第13条 | 隐私协议+勾选确认 |
| 数据加密 | 《数据安全法》第27条 | 敏感字段AES加密 |
| 访问控制 | 《网络安全法》第21条 | RBAC权限模型 |
| 数据删除 | 《个人信息保护法》第47条 | 支持数据删除/匿名化 |
| 跨境传输 | 《个人信息保护法》第38条 | 数据不出境 |

### 5.2 数据留存

| 数据类型 | 留存期限 | 法规依据 |
|----------|----------|----------|
| 保单数据 | 永久 | 《保险法》 |
| 理赔数据 | ≥10年 | 《保险法》 |
| 交易记录 | ≥5年 | 《反洗钱法》 |
| 操作日志 | ≥3年 | 《网络安全法》 |
| 客户录音 | ≥10年 | 《保险销售行为管理办法》 |

## 6. 报送要求

### 6.1 监管报送

| 报送类型 | 频率 | 报送对象 | 系统支持 |
|----------|------|----------|----------|
| 保单登记 | 实时 | 保险保障基金 | 自动报送 |
| 偿付能力报告 | 季度 | 银保监会 | 自动生成 |
| 反洗钱报告 | 月度 | 人民银行 | 自动+人工审核 |
| 消费者投诉 | 月度 | 银保监会 | 统计报表 |
| 重大事项 | 即时 | 银保监会 | 人工触发 |

## 7. 审计要求

| 审计项 | 频率 | 说明 |
|--------|------|------|
| 合规自查 | 季度 | 各部门自查+合规部抽查 |
| 外部审计 | 年度 | 第三方审计机构 |
| 信息安全审计 | 年度 | 等保测评 |
| 反洗钱审计 | 年度 | 专项审计 |
