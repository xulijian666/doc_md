## 2. 文档类型矩阵

```dataview
TABLE length(rows) AS "数量", join(rows.file.link, ", ") AS "文档列表"
FROM ""
WHERE 文档类型 != null
GROUP BY 文档类型
SORT length(rows) DESC
```

---

## 3. 状态看板

```dataview
TASK
FROM ""

```

---

## 4. 高频引用文档

> 被其他文档引用最多的文档，是知识网络的核心节点。

```dataview
TABLE length(file.inlinks) AS "被引用次数", file.inlinks AS "引用来源"
FROM ""
WHERE length(file.inlinks) > 0
SORT length(file.inlinks) DESC
LIMIT 10
```

---

## 5. 孤立文档

> 没有被任何文档引用的文档，可能需要补充关联。

```dataview
TABLE 文档类型 AS "类型", 编制人 AS "编制人"
FROM ""
WHERE length(file.inlinks) = 0 AND 文档类型 != null AND 文档类型 != "看板" AND 文档类型 != "知识图谱"
SORT file.name ASC
```

---

## 6. 全库标签云

```dataview
TABLE length(rows) AS "文档数"
FROM ""
WHERE length(file.tags) > 0
FLATTEN file.tags AS tag
GROUP BY tag
SORT length(rows) DESC
```

---

## 7. 最近修改

```dataview
TABLE 文档类型 AS "类型", 编制人 AS "编制人", file.mtime AS "修改时间"
FROM ""
WHERE 文档类型 != null
SORT file.mtime DESC
LIMIT 10
```

## 8. 双向链接关系图

> 在 Obsidian 中按 `Ctrl+G` 打开图谱视图，可以看到所有文档的链接关系。

```dataview
TABLE file.outlinks AS "链接到", file.inlinks AS "被链接"
FROM ""
WHERE length(file.outlinks) > 0 OR length(file.inlinks) > 0
SORT length(file.outlinks) DESC
LIMIT 15
```
