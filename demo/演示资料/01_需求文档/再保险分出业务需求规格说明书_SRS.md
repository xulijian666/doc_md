## 1. 项目背景

再保险是保险公司分散风险的重要手段。随着业务规模扩大，需要系统化管理分出业务，包括合同分保、临分安排、分保账单等。

## 2. 再保险类型

| 类型 | 说明 | 适用场景 |
|------|------|----------|
| 比例合同分保 | 按固定比例分出保费和赔款 | 常规业务线 |
| 非比例合同分保 | 超过自留额部分分出 | 高保额业务 |
| 临时分保 | 逐笔安排分保 | 大型特殊风险 |
| 预约分保 | 介于合同与临分之间 | 中等风险业务 |

## 3. 核心功能

### 3.1 分保合同管理

#### FR-001 合同配置

| 参数 | 说明 |
|------|------|
| 合同编号 | 系统自动生成 |
| 合同类型 | 比例/非比例 |
| 分出比例 | 1%~100% |
| 自留额 | 公司自留的最大责任额 |
| 分保接受人 | 再保险公司列表 |
| 合同期限 | 起止日期 |
| 手续费比例 | 分保手续费率 |

### 3.2 分保计算

#### FR-002 比例分保计算

```
分出保费 = 原保费 × 分出比例
分出责任 = 原保额 × 分出比例
摊回赔款 = 原赔款 × 分出比例
```

#### FR-003 非比例分保计算

```
分出保费 = 超额赔款再保险费率 × 原保费收入
摊回赔款 = min(实际赔款 - 自留额, 分保限额)
```

### 3.3 分保账单

#### FR-004 账单类型

| 账单类型 | 生成周期 | 内容 |
|----------|----------|------|
| 预付保费账单 | 季度 | 预估保费的分出部分 |
| 调整保费账单 | 年度 | 实际保费与预估的差额 |
| 赔款账单 | 月度 | 当月摊回赔款 |
| 佣金账单 | 季度 | 分保手续费 |

### 3.4 临分管理

#### FR-005 临分申请流程

```
业务部门提交临分申请
    │
    ▼
再保部门审核风险
    │
    ▼
向再保接受人询价
    │
    ├── 接受 ──▶ 确认分保条件 ──▶ 生成临分单
    │
    └── 拒绝 ──▶ 调整条件重新询价/拒绝承保
```

## 4. 报表需求

| 报表 | 频率 | 内容 |
|------|------|------|
| 分保业务汇总表 | 月度 | 各合同分出保费、赔款汇总 |
| 自留额执行情况表 | 月度 | 各业务线自留比例监控 |
| 临分业务明细表 | 实时 | 临分申请、确认、赔付明细 |
| 再保人对账单 | 季度 | 与各再保人的账务核对 |

## 5. 接口需求

| 接口 | 方向 | 用途 |
|------|------|------|
| 核心承保系统 | 入站 | 获取原保单信息 |
| 理赔系统 | 入站 | 获取赔案信息 |
| 财务系统 | 出站 | 分保账务处理 |
| 精算系统 | 出站 | 分保数据提供 |
