## 1. 项目背景

保全业务是寿险售后服务的核心环节，涵盖保单信息变更、缴费管理、退保、给付等全生命周期服务。当前系统存在以下问题：

- 保全项目分散在多个子系统，客户体验割裂
- 部分保全操作仍需线下柜台办理，效率低
- 缺乏统一的保全流程编排能力

## 2. 业务目标

| 目标编号 | 目标描述 | 量化指标 |
|----------|----------|----------|
| BG-01 | 线上化率 | 保全业务线上化率 ≥ 85% |
| BG-02 | 处理时效 | 常规保全 ≤ 1个工作日，复杂保全 ≤ 3个工作日 |
| BG-03 | 差错率 | 保全操作差错率 ≤ 0.1% |
| BG-04 | 客户满意度 | 保全服务NPS ≥ 80 |

## 3. 保全项目分类

### 3.1 常规保全（线上自助）

| 项目编号 | 项目名称 | 说明 | 所需材料 |
|----------|----------|------|----------|
| BQ-001 | 客户信息变更 | 姓名、联系方式、地址等 | 身份证明 |
| BQ-002 | 受益人变更 | 变更身故受益人 | 保单、身份证明 |
| BQ-003 | 缴费方式变更 | 年缴↔月缴、银行卡变更 | 新银行卡信息 |
| BQ-004 | 保单贷款 | 以保单现金价值贷款 | 保单、身份证明 |
| BQ-005 | 红利领取 | 累积红利领取 | 身份证明 |
| BQ-006 | 保单补发 | 保单遗失补发 | 身份证明、声明书 |

### 3.2 复杂保全（需审核）

| 项目编号 | 项目名称 | 说明 | 审核级别 |
|----------|----------|------|----------|
| BQ-101 | 投保人变更 | 变更保单投保人 | 高级审核 |
| BQ-102 | 减额交清 | 不再缴费，降低保额 | 高级审核 |
| BQ-103 | 保单迁移 | 跨机构保单转移 | 中级审核 |
| BQ-104 | 复效 | 失效保单恢复效力 | 高级审核 |
| BQ-105 | 加保 | 增加保额或附加险 | 核保审核 |

### 3.3 退保

| 项目编号 | 项目名称 | 退保类型 | 退费规则 |
|----------|----------|----------|----------|
| BQ-201 | 犹豫期退保 | 全额退保 | 退还全部已交保费（扣除工本费10元） |
| BQ-202 | 正常退保 | 退现金价值 | 退还保单当时现金价值 |
| BQ-203 | 全额退保 | 特殊情况 | 存在销售误导等合规问题 |

## 4. 核心流程

### 4.1 保全申请流程

```
客户发起申请
    │
    ▼
身份验证（短信+人脸识别）
    │
    ▼
选择保全项目
    │
    ▼
填写变更信息
    │
    ▼
上传材料（OCR识别）
    │
    ▼
信息确认与电子签名
    │
    ├── 简单保全 ──▶ 系统自动审核 ──▶ 即时生效
    │
    └── 复杂保全 ──▶ 人工审核 ──▶ 审核通过 ──▶ 生效
                         │
                         └── 审核驳回 ──▶ 通知客户
```

### 4.2 退保流程

#### FR-001 退保试算

- 客户发起退保前可先进行退保试算
- 展示：已交保费总额、当前现金价值、退保手续费、实际退费金额
- 展示退保损失率（现金价值/已交保费）
- 犹豫期内提示可全额退保

#### FR-002 退保审核规则

| 规则 | 条件 | 处理 |
|------|------|------|
| 犹豫期判断 | 签收保单 ≤ 15天 | 走犹豫期退保流程 |
| 贷款抵扣 | 存在未还保单贷款 | 退费金额扣除贷款本息 |
| 理赔关联 | 有在处理的理赔案件 | 暂不允许退保 |
| 团险限制 | 团体保单被保险人 | 需投保人确认 |

#### FR-003 退费支付

- 退费至原缴费账户
- 原账户不可用时，退至投保人本人其他银行卡
- 退费时效：审核通过后 ≤ 5个工作日

### 4.3 保单贷款流程

#### FR-004 贷款规则

| 参数 | 规则 |
|------|------|
| 贷款上限 | 现金价值的 80% |
| 贷款期限 | ≤ 6个月 |
| 贷款利率 | 年化 4.5%（随LPR浮动） |
| 还款方式 | 到期一次性还本付息 |
| 续贷条件 | 利息结清后可续贷 |

## 5. 计算规则

### 5.1 现金价值计算

```
现金价值 = 已交保费 - 保险公司管理费用分摊 - 佣金 - 已承担风险保费 + 利息
```

现金价值表按险种、缴费年限预设，系统自动查表。

### 5.2 退保手续费

| 保单年度 | 手续费比例 |
|----------|-----------|
| 第1年 | 5% |
| 第2年 | 4% |
| 第3年 | 3% |
| 第4-5年 | 2% |
| 第6年及以上 | 0% |

## 6. 非功能需求

| 指标 | 要求 |
|------|------|
| 保全申请响应时间 | ≤ 2s |
| 退保试算响应时间 | ≤ 3s |
| 人脸识别比对时间 | ≤ 5s |
| 日处理保全量 | ≥ 3万笔 |
