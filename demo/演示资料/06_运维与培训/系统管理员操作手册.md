## 2. 产品配置

### 2.1 新增产品

```
1. 进入【产品管理】→【产品列表】
2. 点击【新增产品】
3. 填写产品基本信息：
   - 产品编码（自动生成）
   - 产品名称
   - 产品类型（车险/健康险/寿险/财产险）
   - 产品类别（主险/附加险）
   - 投保年龄范围
4. 配置保障项：
   - 添加保障项
   - 设置保额范围
   - 设置免赔额
   - 设置赔付比例
5. 关联费率表
6. 提交审核
7. 审核通过后上架
```

### 2.2 产品上下架

| 操作 | 条件 | 影响 |
|------|------|------|
| 上架 | 产品审核通过 | 新投保可选择该产品 |
| 下架 | 无在途投保单 | 停止新投保，已有保单不受影响 |
| 停售 | 提前30日报备 | 停止新投保+续保 |

## 3. 费率管理

### 3.1 费率表配置

```
1. 进入【费率管理】→【费率表】
2. 选择关联的保障项
3. 配置费率因子：
   - 年龄段费率
   - 性别费率
   - 地区费率
   - NCD系数表
4. 设置生效日期
5. 提交精算审核
6. 审核通过后生效
```

### 3.2 费率调整

| 调整类型 | 审批要求 | 生效时间 |
|----------|----------|----------|
| 常规调整 | 精算师+合规部 | 下一保单年度 |
| 临时调整 | CTO审批 | 即时 |
| 监管要求 | 合规部确认 | 按监管要求 |

## 4. 核保规则配置

### 4.1 规则引擎管理

```
1. 进入【核保管理】→【规则引擎】
2. 选择规则类型（自动核保/人工核保）
3. 配置规则条件：
   - 字段选择（如：车龄、出险次数）
   - 运算符（大于、小于、等于、包含）
   - 阈值设置
4. 配置规则动作：
   - 通过
   - 拒绝
   - 转人工
   - 附加条件
5. 设置规则优先级
6. 测试规则
7. 发布规则
```

### 4.2 常用规则示例

| 规则 | 条件 | 动作 |
|------|------|------|
| 高龄车辆 | 车龄 > 15年 | 拒绝车损险 |
| 高风险车型 | 品牌在黑名单 | 转人工核保 |
| 短期投保 | 保险期间 < 1年 | 转人工核保 |
| 高频出险 | 近3年出险 ≥ 5次 | 转人工核保 |
| 信用不良 | 信用分 < 600 | 拒绝承保 |

## 5. 渠道管理

### 5.1 渠道接入配置

```
1. 进入【渠道管理】→【渠道列表】
2. 点击【新增渠道】
3. 填写渠道信息：
   - 渠道编码
   - 渠道名称
   - 对接模式（API/H5/小程序）
   - 联系人信息
4. 配置API参数：
   - 生成AppID和AppSecret
   - 配置回调地址
   - 配置IP白名单
5. 配置产品权限
6. 配置费率系数
7. 测试环境联调
8. 正式上线
```

### 5.2 API密钥管理

| 操作 | 说明 | 安全要求 |
|------|------|----------|
| 生成密钥 | 新渠道接入时生成 | 仅显示一次，需妥善保存 |
| 重置密钥 | 密钥泄露时重置 | 旧密钥立即失效 |
| 停用密钥 | 渠道停用时 | 已有交易不受影响 |

## 6. 报表管理

### 6.1 报表订阅

```
1. 进入【报表中心】→【报表订阅】
2. 选择报表类型
3. 配置订阅参数：
   - 接收人
   - 发送频率（日报/周报/月报）
   - 发送方式（邮件/钉钉）
4. 保存订阅
```

### 6.2 自定义报表

```
1. 进入【报表中心】→【自定义报表】
2. 选择数据源
3. 配置维度和指标
4. 配置筛选条件
5. 选择展示方式（表格/图表）
6. 保存并分享
```

## 7. 系统参数配置

### 7.1 可配置参数

| 参数 | 说明 | 默认值 | 修改影响 |
|------|------|--------|----------|
| 犹豫期天数 | 退保全额退款期限 | 15天 | 影响退保规则 |
| 自动核保超时 | 转人工核保等待时间 | 1分钟 | 影响核保时效 |
| 报价有效期 | 报价结果有效时长 | 7天 | 影响报价使用 |
| 验证码有效期 | 短信验证码有效时长 | 5分钟 | 影响登录/验证 |
| 密码错误锁定 | 连续错误锁定次数 | 5次 | 影响账户安全 |
| 会话超时 | 登录会话有效期 | 30分钟 | 影响用户体验 |

### 7.2 参数修改流程

```
1. 提交参数修改申请
2. 说明修改原因和影响
3. 部门经理审批
4. 测试环境验证
5. 生产环境执行
6. 记录变更日志
```

## 8. 常见问题处理

| 问题 | 原因 | 处理方式 |
|------|------|----------|
| 报价失败 | 费率表未配置 | 检查费率表配置 |
| 核保超时 | 规则引擎异常 | 检查规则配置+日志 |
| 支付失败 | 渠道接口异常 | 检查渠道配置+联系渠道方 |
| 保单生成失败 | 模板配置错误 | 检查保单模板 |
| 通知发送失败 | 消息通道异常 | 检查短信/邮件配置 |
