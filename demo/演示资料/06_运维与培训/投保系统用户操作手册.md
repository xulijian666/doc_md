## 1. 客户端操作

### 1.1 车险投保流程

**Step 1: 进入投保页面**
- 打开APP → 首页 → 【车险投保】
- 或扫描代理人分享的二维码

**Step 2: 填写车辆信息**
```
1. 输入车牌号（支持拍照识别）
2. 输入车架号VIN（支持OCR识别行驶证）
3. 选择品牌型号（模糊搜索）
4. 选择初次登记日期
5. 选择使用性质
6. 确认座位数
7. 点击【下一步】
```

**Step 3: 填写投保人信息**
```
1. 输入投保人姓名
2. 选择证件类型
3. 输入证件号码（支持OCR识别）
4. 输入手机号
5. 获取并输入验证码
6. 填写联系地址
7. 点击【下一步】
```

**Step 4: 选择险种**
```
1. 勾选交强险（必选）
2. 勾选需要的商业险种：
   - 机动车损失险
   - 第三者责任险（选择保额：50万/100万/150万/200万）
   - 车上人员责任险（驾驶员/乘客）
   - 附加险（涉水险、划痕险等）
3. 点击【获取报价】
```

**Step 5: 确认报价**
```
1. 查看保费明细
2. 查看各险种保额和保费
3. 查看费率系数说明
4. 确认无误后点击【立即投保】
```

**Step 6: 支付**
```
1. 选择支付方式（微信/支付宝/银行卡）
2. 完成支付
3. 等待出单（通常30秒内）
```

**Step 7: 查看保单**
```
1. 支付成功后自动跳转保单详情
2. 可下载电子保单PDF
3. 保单同时发送至APP消息、短信、邮箱
```

### 1.2 健康险投保流程

**Step 1: 选择产品**
- 进入【健康险】→ 选择产品（百万医疗/重疾险等）

**Step 2: 填写被保险人信息**
```
1. 填写姓名、证件号、手机号
2. 选择职业类别
3. 确认投保年龄符合要求
```

**Step 3: 健康告知**
```
1. 逐条阅读健康告知问题
2. 如实选择"是"或"否"
3. 选择"是"时需填写详细说明
4. 确认提交
```

**Step 4: 选择方案**
```
1. 选择保额
2. 选择有无社保
3. 确认保费
```

**Step 5: 支付出单**
- 同车险流程

## 2. 代理人操作

### 2.1 代客投保

```
1. 登录代理人工作台
2. 点击【代客投保】
3. 选择产品类型
4. 代客户填写信息（同客户操作）
5. 客户确认（短信验证码/电子签名）
6. 完成投保
```

### 2.2 保单管理

| 功能 | 操作路径 | 说明 |
|------|----------|------|
| 保单查询 | 保单管理 → 保单列表 | 查看名下所有保单 |
| 保单详情 | 点击保单号 | 查看保单详细信息 |
| 续保提醒 | 保单管理 → 待续保 | 查看即将到期保单 |
| 佣金查询 | 佣金管理 → 佣金明细 | 查看佣金收入 |

## 3. 核保员操作

### 3.1 待核保任务处理

```
1. 登录管理后台
2. 进入【核保管理】→【待核保任务】
3. 查看任务列表（按风险等级排序）
4. 点击任务进入详情页
5. 查看：
   - 投保人/被保险人信息
   - 车辆信息（车险）/ 健康告知（健康险）
   - 风险评分和风险因子
   - 历史投保/理赔记录
6. 做出决策：
   - 【通过】：同意承保
   - 【拒绝】：选择拒保理由
   - 【条件承保】：附加承保条件
7. 填写审核意见
8. 提交
```

### 3.2 核保时效管理

| 风险等级 | 时效要求 | 超时处理 |
|----------|----------|----------|
| 高风险 | 2小时内 | 自动升级至主管 |
| 中风险 | 4小时内 | 系统提醒 |
| 低风险 | 8小时内 | 系统提醒 |

## 4. 理赔员操作

### 4.1 案件受理

```
1. 登录管理后台
2. 进入【理赔管理】→【待受理案件】
3. 选择案件查看详情
4. 核实报案信息
5. 检查材料完整性：
   - 身份证 ✓
   - 银行卡 ✓
   - 医疗发票 ✓
   - 病历 ✓
   - 费用明细 ✓
6. 材料齐全 → 受理案件
7. 材料不全 → 通知客户补充
```

### 4.2 案件审核

```
1. 进入【理赔管理】→【待审核案件】
2. 查看OCR识别结果
3. 核对医疗费用明细
4. 查看风险评分
5. 查看保单条款和免责条款
6. 做出决策：
   - 【通过】：确认赔付金额
   - 【部分通过】：调整赔付金额
   - 【拒赔】：选择拒赔理由（关联条款）
7. 提交审核
```

## 5. 保全操作

### 5.1 客户自助保全

| 操作 | 路径 | 说明 |
|------|------|------|
| 信息变更 | 我的保单 → 保全 → 信息变更 | 修改联系方式、地址等 |
| 受益人变更 | 我的保单 → 保全 → 受益人变更 | 需人脸识别验证 |
| 缴费方式变更 | 我的保单 → 保全 → 缴费变更 | 更换扣款银行卡 |
| 退保 | 我的保单 → 保全 → 退保 | 展示退保损失，需确认 |
| 保单贷款 | 我的保单 → 保全 → 保单贷款 | 展示可贷金额 |

### 5.2 退保操作流程

```
1. 选择要退保的保单
2. 点击【退保】
3. 系统展示退保试算：
   - 已交保费总额
   - 当前现金价值
   - 退保手续费
   - 实际退费金额
   - 退保损失率
4. 确认退保（需人脸识别）
5. 选择退费账户
6. 提交退保申请
7. 等待审核（1-3个工作日）
8. 审核通过后退费到账（5个工作日内）
```

## 6. 常见问题FAQ

| 问题 | 解答 |
|------|------|
| 报价后多久有效？ | 报价结果7天内有效 |
| 支付失败怎么办？ | 可重新进入投保单继续支付，报价仍有效 |
| 保单何时生效？ | 支付成功后即时生效 |
| 如何下载电子保单？ | 我的保单 → 选择保单 → 下载电子保单 |
| 忘记保单号怎么办？ | 使用投保人证件号+手机号查询 |
| 退保多久到账？ | 审核通过后5个工作日内 |
| 如何变更受益人？ | 我的保单 → 保全 → 受益人变更 |
| 理赔多久能结案？ | 小额案件1个工作日，大额案件3-5个工作日 |
