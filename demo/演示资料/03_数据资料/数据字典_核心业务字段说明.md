## 4. 保单相关字段

| 字段编码 | 字段名称 | 数据类型 | 取值说明 | 业务含义 |
|----------|----------|----------|----------|----------|
| POLICY_NO | 保单号 | varchar(32) | 见保单号规则 | 保单唯一标识 |
| PROPOSAL_NO | 投保单号 | varchar(32) | P+日期+序列号 | 投保申请单号 |
| POLICY_STATUS | 保单状态 | varchar(20) | 见保单状态枚举 | 保单当前状态 |
| TOTAL_PREMIUM | 总保费 | decimal(15,2) | 单位：元 | 应缴总保费 |
| PAID_PREMIUM | 已缴保费 | decimal(15,2) | 单位：元 | 实际已缴金额 |
| CASH_VALUE | 现金价值 | decimal(15,2) | 单位：元 | 退保可退金额 |
| EFFECTIVE_DATE | 生效日期 | date | yyyy-MM-dd | 保险责任开始日 |
| EXPIRE_DATE | 到期日期 | date | yyyy-MM-dd | 保险责任终止日 |

### 保单号规则

```
格式：{地区码(2)}{险种码(2)}{年份(4)}{序列号(10)}
示例：
  BJ-CAR-2026-0000000001  北京-车险-2026-第1号
  SH-HL-2026-0000000001   上海-健康险-2026-第1号
  GD-LF-2026-0000000001   广东-寿险-2026-第1号
```

### 保单状态枚举

| 状态码 | 名称 | 说明 |
|--------|------|------|
| DRAFT | 草稿 | 投保单创建，未提交 |
| QUOTED | 已报价 | 费率计算完成 |
| SUBMITTED | 已提交 | 等待核保 |
| UNDERWRITING | 核保中 | 自动/人工核保进行中 |
| APPROVED | 核保通过 | 等待支付 |
| REJECTED | 核保拒绝 | 不可承保 |
| PAID | 已支付 | 保费已收取 |
| ISSUED | 已出单 | 保单号已生成 |
| EFFECTIVE | 已生效 | 保险责任开始 |
| LAPSED | 失效 | 未续保或欠费 |
| SURRENDERED | 已退保 | 退保完成 |
| EXPIRED | 已过期 | 保险期满 |
