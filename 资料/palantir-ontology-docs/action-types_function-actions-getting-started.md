# 入门

本教程解释了如何创建一个由Ontology 编辑函数支持的操作类型。

## 先决条件

在本教程中，我们将使用与操作入门教程中相同的Demo Ticket对象类型和示例对象。

首先编写一个Ontology 编辑函数，以执行操作所需的更改。这需要：

- 使用TypeScript模板在对象上设置一个函数存储库，
- 将相关对象类型导入到您的存储库中，并且
- 发布Ontology 编辑函数以供操作读取。

有关这些步骤的信息可以在函数文档中找到：

- 入门——遵循本教程创建一个基本的函数存储库并发布一个函数
