# 存根对象搜索和聚合

在编写单元测试时，您可能希望为对象集搜索或对象聚合创建预设答案（也称为“存根”），以指定您的代码在编写单元测试时所做调用的响应。您需要从"@foundry/functions-testing-lib"中导入{ whenObjectSet }以使用存根。

#### 测试对象集上的筛选

```
1
2
3
4
5
6
7

```

#### 使用存根测试对象属性上的聚合

您可以使用存根定义对聚合调用的响应。

```
1
2
3
4
5

```

这意味着每当运行Objects.search().objectType().sum(s => s.property))时，结果将是55。

#### 使用存根测试对象

您还可以使用存根定义某些对象搜索的响应。

```
1
2
3
4
5
6
7

```

这意味着，每当运行这个特定的对象搜索聚合时，属性和将解析为65。
