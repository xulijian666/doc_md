# 编辑结构属性类型

结构可用性

结构属性类型目前正在开发中，将于2024年9月全面推出。

1. 在Ontology Manager中，打开左侧边栏的Object类型选项卡，并选择一个现有的Object类型。
2. 在Object类型详情页面中，打开左侧边栏的属性选项卡，并从属性表中选择相关的结构属性类型。
3. 在属性编辑器面板中，滚动到结构字段部分并选择您想要编辑的结构字段。编辑次数将显示在Ontology Manager界面的右上角。

![‘属性编辑器’面板中的结构字段。](/docs/resources/foundry/object-link-types/Edit-struct.png?width=500)
1. 在编辑结构字段对话框中进行必要的编辑，然后选择确认。

![‘编辑结构字段’对话框。](/docs/resources/foundry/object-link-types/confirm-struct-edit.png?width=500)
