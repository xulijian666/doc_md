# 概述

有几个基本概念对于Foundry Rules至关重要：

- Object模型：Foundry规则是具有属性的Objects，可以通过规则提案进行管理。
- Workshop应用程序：Foundry规则由Workshop应用程序管理，通常包括规则编辑器和提案审阅器功能。
- 规则逻辑：Foundry规则编码旨在执行特定工作流程的业务逻辑。此逻辑遵循特定结构，并定义了一组数据变换。
- Foundry Rules工作流程配置：使用工作流程配置编辑器配置Foundry规则。编辑器是工作流程所有者用来将Foundry Rules定制到他们的应用案例的界面。

在2022年7月之前，Foundry Rules包含了额外的概念并需要更多配置。如果您在此日期之前部署了Foundry Rules，请了解这些额外概念和配置。
