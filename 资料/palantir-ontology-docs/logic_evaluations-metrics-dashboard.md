# 评估指标仪表盘

评估套件运行的指标被收集在报告中，可以在评估指标仪表盘中查看。您可以比较评估函数的汇总结果和/或检查单个测试用例的结果。

要进行更深入的分析、查看LLM踪迹或比较不同运行之间的结果，请在逻辑函数视图中选择查看指标仪表盘。从那里，您可以选择批次运行，以深入比较指标、持续时间和其他基准。
