# 概述

Vertex 包含一套功能，旨在帮助您了解您的组织如何随时间变化。这对于分析过去的事件和预测未来的事件特别有帮助，尤其是结合 Vertex 对情景的支持。

您应该熟悉两种主要的时间数据类型：事件和时间序列。

- 事件是本体中配置的Object类型，包含时间信息——至少两个时间戳，表示事件的开始和结束。
- 时间序列表示随时间测量的值，可以在本体中配置为时间序列属性。

通过学习如何配置事件来开始，然后了解如何在 Vertex 中使用事件和时间序列：

- 探索相关事件
- 探索相关时间序列
- 使用时间选择
- 配置阈值
