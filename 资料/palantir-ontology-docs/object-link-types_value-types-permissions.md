# 值类型权限

值类型的权限管理通过平台空间进行。空间中的任何值类型都会自动导入并可用于关联的Ontology。其他使用者可以将值类型导入到他们的项目范围中，就像用户可以导入变换配置文件或管道输入一样。

具有某个空间查看（读取）权限的用户可以将值类型指派给该空间和关联Ontology中的属性类型或共享属性类型。空间的编辑者或所有者可以在该空间中创建、编辑或删除值类型。
