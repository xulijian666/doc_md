# 配置链接合并

在Ontology管理器的Vertex Ontology功能部分配置链接合并将允许Object自动分组到两个节点之间的边缘。此配置有助于可视化节点之间的事务流并简化图的生成。

用户可以通过搜索周围菜单中的通过链接选项访问此功能。
