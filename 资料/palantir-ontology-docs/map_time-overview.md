# 地图中的时间和时间数据

地图具有一系列用于可视化和处理随时间变化数据的功能。时间数据有多种形式，每种形式都可以以不同的方式使用和可视化。地图应用程序中显示的所有时间数据都遵循当前选择的时间，以帮助您了解数据随时间的变化并检查过去的特定时间。

## 时间序列

时间序列是随时间变化的测量值。您可以在Ontology中将时间序列值配置为时间序列属性。使用时间序列在地图上样式化Objects，并在序列面板中查看它们。

## 事件

事件是与特定时间或时间范围关联的具有附加元数据的Objects。事件可以被用于控制Objects的不透明度在您的地图上，并在时间轴中可视化。

## 轨迹

使用轨迹来表示位置随时间变化的Objects。轨迹样式选项允许您自定义Objects随时间变化的位置的可视化方式。
