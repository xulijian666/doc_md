# 图层管理

地图上的所有数据都被分组到不同的图层类型中，可以在图层面板中进行管理。

## 切换图层可见性

通过使用可见性切换来显示或隐藏图层的内容：

![隐藏图层按钮](/docs/resources/foundry/map/layer-management-hide-layer.png)

## 重命名图层

通过点击当前名称来编辑图层的名称。

![编辑图层名称](/docs/resources/foundry/map/layer-management-edit-layer-name.png)

## 重新排序图层

通过拖动图层图标来更改图层的顺序。重新排序图层可以改变地图的渲染效果，因为在图层列表中出现较高的图层会渲染在列表中较低的图层之上。

## 移动对象到新的或现有的图层
