# 配置链式模型

当您开始在整个建模宇宙中连接系统和流程时，您必须能够理解系统中每个单独方面如何在全局范围内交互。Vertex允许您将连接的模型链在一起，以理解和量化更改的端到端影响，从而以实现最佳决策。

一旦您将模型添加到案例研究中，您可以选择+ 添加新模型以搜索和选择其他相关模型。任何为模型输出生成的值，如果被映射到参数并用作后续模型的输入，将作为该模型的输入值传播。在输入单元格中使用链接图标标识此值。

所选模型将被添加到情景窗格中，您可以在案例研究中选择显示输入/输出参数。这些是按模型显示的。

在任何选定模型输入中添加覆盖，将在您运行“假设”模拟时显示受影响的输入，并且如果启用了该选项，将触发运行基线场景。
