# 启用非必填功能

通过编辑规则工作坊应用程序中规则编辑器微件的配置，可以启用或禁用非必填功能，如下图所示。

Foundry Rules 可以启用或禁用一系列非必填逻辑面板：

- 窗口面板：支持窗口函数 ↗。
- 聚合面板：计算分组列上的聚合。
- 合并面板：合并其他数据集或对象。
- 表达式面板：执行任意表达式以添加列或筛选。
- 选择列面板：选择部分列以传递到下一个逻辑面板。
- 联合面板：联合其他数据集或对象。

此外，还有一个选项可以启用或禁用从 Contour 导入规则。

- Contour 导入：导入并转换存储在Contour 分析中的逻辑为规则。

最后，Foundry Rules 支持直接在时间序列数据上编写规则。

- 时间序列：添加时间序列面板，可以直接作为规则的一部分操作时间序列。
