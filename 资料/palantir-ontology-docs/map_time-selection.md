# 时间选择

使用时间选择面板，您可以调整当前选择的时间和时间窗口。通过滚动时间窗口，您可以了解数据随时间的变化。

## 更改选择的时间

使用时间选择器调整您的地图，以在感兴趣的时间可视化数据。

时间选择器仅允许选择位于当前时间窗口内的时间。

您也可以使用滑块滚动您的时间窗口：

### 查看最新数据

点击查看最新将所选时间设置为当前时间。在最新数据视图中，所选时间会自动更新以匹配当前时间。将最新数据模式与流数据结合使用，以在您的地图上可视化实时更新的数据。
