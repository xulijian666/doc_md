# 编写并运行规则

以下步骤将指导您在Workshop应用程序中编写和运行规则的过程。

1. 找到Workshop Rule应用程序：从工作流配置屏幕中，选择文件夹并选择Workshop应用程序。

1. 编写规则：在上一步创建的Workshop应用程序中，点击创建新建按钮开始创建规则。(a) 在规则顶部的表单中填写名称、描述和其他信息。(b) 编写您希望规则执行的逻辑。例如，这可以是一个简单的筛选。(c) 点击提交更改以为此新规则创建一个提案。
2. 批准提案：在Workshop应用程序的提案选项卡内，选择左侧新创建的提案。选择批准以激活它作为规则。
3. 搭建规则数据输出和规则输出数据集：导航到配置工作流时创建的输出数据集。选择操作，然后探索数据沿袭以查看输入数据集。选择规则数据输出数据集(d)和输出数据集(e)。右键点击选择的两个数据集并选择搭建。一旦搭建完成，输出数据集将包含您新规则的结果。未来，这两个数据集可以被放置在一个计划上以保持输出的更新。
