# 入门

Logic运行面板非常适合一次性测试，但要对Logic函数有更大的信心，重要的是针对多个输入进行测试。Evaluations使得搭建严格的测试和评估套件变得容易。请按照以下步骤开始为Logic函数使用Evaluations：

1. 在添加测试用例或配置评估套件之前，保存您的Logic函数。
2. 在Logic运行面板中选择您想要用于测试用例的输入，然后在底部选择添加为测试用例。这将使用当前输入生成一个测试用例。

1. 选择右侧工具栏上的锤子图标以查看所有先前创建的测试用例。
2. 一旦添加了所需数量的测试用例，选择运行评估套件。如需其他运行时选项，选择运行评估套件右侧的齿轮图标以打开运行配置对话框。在这里，您可以选择要使用的Logic版本、每个测试运行的次数，或为您的套件运行添加元数据。

1. 一旦您的套件运行完毕，运行摘要将显示在最近结果部分。选择眼睛图标将打开结果预览对话框，您可以在其中查看每个测试用例的结果。这包括函数输出、持续时间和每个测试用例的成本。结果预览对话框还允许您比较两个不同的评估套件运行。
