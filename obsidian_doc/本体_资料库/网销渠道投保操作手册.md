# 网销渠道投保操作手册

## 1. 适用范围

本手册适用于通过公司官方APP、微信小程序销售的人身保险产品，当前可售产品为[[少儿重大疾病保险（2026版）产品条款]]（少儿重大疾病保险2026版）。

## 2. 投保流程

### 2.1 产品展示

投保人在前端页面选择产品后，系统展示产品详情页，包括保险责任说明、费率试算结果、免责条款提示。页面须以加粗方式提示投保人仔细阅读保险条款。

### 2.2 信息填写

投保人填写以下信息：

- 投保人基本信息（姓名、证件号、联系方式）
- 被保险人基本信息（姓名、证件号、职业类别）
- 受益人信息（指定受益人或法定受益人）
- 缴费方式选择（年缴/月缴）

系统须对证件号进行实名校验，调用公安实名认证接口完成身份核验。

### 2.3 健康告知

系统以逐题问答方式向投保人展示健康告知问卷，投保人须如实作答。健康告知存在异常的，系统自动进入智能核保流程，可能的结论包括标准体承保、加费承保、除外承保、拒保。

### 2.4 缴费与出单

核保通过后生成缴费订单，支持微信支付、支付宝、银行卡快捷支付。缴费成功后系统签发电子保单。

## 3. 特殊场景

### 3.1 高龄投保

被保险人年龄超过60周岁时，系统须强制要求上传体检报告并转人工核保。

### 3.2 大额保件

单张保单累计保额超过100万元时，须触发财务核保，评估投保人的保费承受能力。

### 3.3 重复投保检测

系统须在投保提交前检测被保险人是否已持有同类保单，评估累计保额是否超过承保上限。
