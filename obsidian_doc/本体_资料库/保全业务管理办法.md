# 保全业务管理办法

## 1. 概述

保全指保单生效后的各类变更维护操作，是与理赔并列的两大核心业务之一。

## 2. 保全业务分类

### 2.1 信息变更

投保人联系方式、被保险人基本信息的更正，以及受益人的增加、删除或比例调整。受益人变更须经被保险人同意。

### 2.2 退保

犹豫期退保和正常退保的退费规则及操作流程详见[[退保与退费处理办法]]。

### 2.3 复效

保单中止后两年内可申请复效。复效须满足以下条件：

- 补缴全部欠交保费及利息
- 中止超过60天的须重新进行健康告知和核保
- 复效后重新计算等待期

保单中止的原因及宽限期管理详见[[续期保费催缴与宽限期处理办法]]。

### 2.4 保单贷款与还款

投保人可以保单现金价值为基数申请贷款，贷款金额不超过现金价值的80%。贷款须在保单有效期内还清。

## 3. 操作规范

- 所有保全操作须核实投保人身份
- 操作完成后及时通知投保人（短信/APP推送）
- 保全操作记录留存备查
