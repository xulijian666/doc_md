# 理赔报案与调查手册

## 1. 报案受理

被保险人或受益人可通过客服热线（95XXX，7×24小时）、APP自助报案、柜面报案。报案时采集以下信息：

- 保单号（可通过投保人姓名+证件号查询）
- 出险人姓名、出险时间、出险地点
- 出险经过简述
- 就诊医院（如有）

报案须在事故发生后10日内完成。

## 2. 立案审核

- 核实保单状态为有效（非中止或终止）
- 确认出险时间在保险期间内且已过等待期
- 符合条件的案件正式立案，分配理赔调查人员

## 3. 理赔调查

### 3.1 调查内容

- 核实事故真实性
- 确认是否属于保险责任范围
- 回溯投保时的健康告知是否完整
- 排除保险欺诈嫌疑

### 3.2 调查方式

- 医院病历调阅
- 公安机关事故证明核实
- 被保险人面访
- 第三方调查机构协助

### 3.3 调查时限

简单案件5个工作日内完成，复杂案件30个工作日，需延长的最长可延至60个工作日，须向申请人说明原因。

## 4. 理算与给付

根据[[少儿重大疾病保险（2026版）产品条款]]中的保险责任计算赔付金额。本产品为定额给付型，确诊即赔，按基本保额给付，无免赔额。少儿特定疾病额外给付50%保额。

理赔核定通过后10个工作日内完成给付，保险金支付至受益人指定账户。

## 5. 特殊场景

保单处于宽限期时发生保险事故，保险公司仍承担保险责任，但须从赔付金额中扣除欠交保费。
