# 退保与退费处理办法

## 1. 业务分类

退保分为犹豫期退保和正常退保两种，适用不同的退费计算规则。

## 2. 犹豫期退保

投保人签收保单后15天内（犹豫期）申请退保：

- 退费金额 = 已缴保费 - 工本费（不超过10元）
- 受理后5个工作日内完成退费
- 退费至投保人原缴费账户

## 3. 正常退保

超过犹豫期后的退保，退还保单的现金价值。以[[少儿重大疾病保险（2026版）产品条款]]中的现金价值表为例，0岁男孩投保第1年退保仅能拿回180元，而年缴保费为1280元，损失较大。业务人员须向投保人充分告知退保损失。

## 4. 操作流程

1. 投保人通过APP或柜面提交退保申请
2. 核实投保人身份及保单状态
3. 系统查询现金价值，计算退费金额
4. 如保单存在保单贷款未还清，须先扣除贷款本息
5. 财务岗执行退费
6. 保单状态变更为"退保终止"

## 5. 注意事项

- 退保后保单终止，不可复效，如需保障须重新投保
- 团体保单的退保须由投保单位统一申请
- 涉及身故受益人的保单，退保须经所有受益人书面同意
- 退保金额超过5万元的须进行反洗钱客户身份重新识别
